🔑 设置 GitHub Token

需要在 GitHub 生成具有 repo 权限的 Token。
您也可以直接将 Token 写入 index.html 的 GITHUB_CONFIG.token 中。

如何获取 Token：
1. 访问 GitHub → Settings → Developer settings → Personal access tokens
2. 点击 "Generate new token (classic)"
3. 勾选 repo 权限
4. 生成后复制 Token 粘贴到上方
